◆ processHttpRequest()

std::string HttpTransaction::processHttpRequest ( std::string & recvData,
int clientSocketFd )

HTTPリクエストを処理し、対応するHTTPレスポンスを生成します。

initializeRequestを呼び出してリクエストデータを初期化した後、
HttpWorkflowManagerを使用してHTTP処理の各ステップを実行します。 処理が完了すると、レスポンス文字列を返します。
