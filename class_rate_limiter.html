bool RateLimiter::isLimited ( )
virtual

現在のリクエストレートが設定された制限を超えているかどうかを判定します。

現在の秒数を基にして、その秒に対するリクエスト数が設定された最大リクエスト数を超えているかをチェックします。

Returns
リクエスト数が最大数を超えていればtrue、そうでなければfalse。
Note
要件外の機能。負荷テスト時の挙動チェック用。
