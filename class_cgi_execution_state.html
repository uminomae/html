void CgiExecutionState::setupEnvironmentVariables ( std::vector< std::string > & envStr)
protected

CGIスクリプトの実行環境をstd::string型のvectorでセットアップします。

Parameters
httpTransactionHttpTransactionオブジェクトへの参照。
envStr環境変数を格納する文字列のベクター。
Note
  • Queryを環境変数で渡します。
  • Bodyを親プロセスのwrite()から子プロセスでdup2した標準入力に渡します。
  • Bodyのlengthを環境変数に指定します。
  • ブラウザからのリクエストにContentTypeが指定されない場合、"application/x-www-form-urlencoded"を設定します。
  • SCRIPT＿NAMEは【実行されているスクリプトの絶対パス】です。※【ブラウザから送られてきた要求のURI】とは異なります。
  • SERVER_NAME はクライアントリクエストにあるヘッダー HOST: の値をセットします。
    If the request includes a message-body, the CONTENT_TYPE variable is set to the Internet Media Type [6] of the message-body.
    CONTENT_TYPE = "" | media-type
    media-type = type "/" subtype *( ";" parameter )
    The SCRIPT_NAME variable MUST be set to a URI path
    The SERVER_NAME variable MUST be set to the name of the server host to which the client request is directed
    クライアント要求の宛先となるサーバーホストの名前に設定する必要があります。

◆ waitPidWNOHANG()

int CgiExecutionState::waitPidWNOHANG ( pid_t pid,
int(&) pipefd[2] )
protected

フォークされたプロセスの終了を非ブロッキングで待ちます。

WNOHANGオプションを使用して、子プロセスの終了を待ちます。

Parameters
httpTransactionHttpTransactionオブジェクトへの参照。
pidフォークされたプロセスのPID。
pipefdパイプのファイルディスクリプタ配列。
Returns
子プロセスが正常に終了した場合は0、そうでない場合は1を返します。
Note

  • 子プロセスがまだ終了していない場合でも、親プロセスはブロックされずに直ちに制御を戻します。これにより、親プロセスは他の処理を続けながら定期的に子プロセスの状態をチェックすることができます。
