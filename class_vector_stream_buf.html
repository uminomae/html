Detailed Description

std::vector<char> をストリームバッファとして使用するためのクラス。

このクラスは、std::streambuf を継承し、std::vector<char> に基づくカスタムストリームバッファの実装を提供します。 これにより、std::vector<char> に格納されたデータを std::istream や std::ostream として扱うことができます。
