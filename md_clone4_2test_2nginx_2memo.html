fcgiwrap

  • install brew install fcgiwrap
  • パスを見つける find /usr/local -name fcgiwrap
  • ~./zshrcに登録 ‘echo 'export PATH="/usr/local/sbin:$PATH"’ >> ~/.zshrc
  • 再読み込み source ~/.zshrc`
  • socket パーミッション確認 ls -l /tmp/fcgiwrap.socket
  • socket パーミッション設定　error logでソケットのパーミッションの問題が出たら　詳細下記 sudo chmod 766 /tmp/fcgiwrap.socket
  • 終了 sudo pkill fcgiwrap
  • 起動 sudo fcgiwrap -s unix:/tmp/fcgiwrap.socket &
  • 動作中であることの確認 ps aux | grep fcgiwrap
  • ソケット削除 sudo rm /tmp/fcgiwrap.socket
  • vscodeで.conf編集
    • Findeメニューの「移動」から「フォルダへ移動」 /usr/local/etc/nginx

nginx

  • 停止 sudo nginx -s stop
  • 起動 sudo nginx
  • 再読み込み sudo nginx -s reload
  • 動作中であることの確認 ps aux | grep nginx
  • 動作中であることの確認 ps aux | grep fcgiwrap
  • error log の場所 /usr/local/Cellar/nginx/1.25.3/logs/error.log
    • unix:/tmp/fcgiwrap.socket failed (13: Permission denied) while connecting to upstream
      • fcgiwrapのソケットのパーミッションを666にする
    • 403
      • sudo chmod 755 /Users から全ての通り道に755を設定
  • pidを探す
    • sudo find / -name nginx.pid
