void ParsingMultiPartState::parseMultiPartFormData ( SocketState::ContentDisposition & contentDisposition,
std::vector< char > & boundary,
std::istream * reqIss )

マルチパートフォームデータの解析を行います。

HTTPリクエストのボディからマルチパートフォームデータを解析し、コンテンツディスポジション情報（ファイル名など）やアップロードされたファイルの本体を抽出します。ヘッダー部分とボディ部分を区別して解析し、バウンダリ文字列を使用して各パートを識別します。

Parameters
contentDispositionコンテンツディスポジション情報を格納するための構造体。
boundaryマルチパートフォームデータのバウンダリ文字列。
reqIssHTTPリクエストのボディを表す入力ストリーム。
Note
  • バウンダリ文字列の削除のために、特定の調整（サイズから -1 が必要)を行なっています。
  • そうしないとバイナリデータをテキストエディタで見た場合に、空行がところどころ追加されてしまい、うまく動かないため。
    • その空行は削除する場所で起きているかのような位置でした
