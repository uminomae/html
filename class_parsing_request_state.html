◆ parseExtractDeleteFileName()

void ParsingRequestState::parseExtractDeleteFileName ( )
protected

HTTPクエリ文字列から"delete_file_name="の後ろの部分を抽出し、 _socketState._deleteFileNameに格納します。 また、_socketState._isDeleteMethodフラグを設定します。

HTTPクエリ文字列内の "delete_file_name=" の後ろの部分を検索し、 "&" または末尾までの部分を _socketState._deleteFileName に格納します。

Note
  • DELETEメソッドの指定方法がわからないので、 .html の form による下記のリクエスト限定で対応しています。
    _parsedQuery = delete_file_name=pre.txt
