特定のHTTPヘッダーを処理し、関連する情報を格納します。

Parameters
httpTransactionHTTPトランザクションオブジェクト
keyヘッダーキー
valueヘッダーの値

この関数は、特定のHTTPヘッダーを処理し、関連する情報をHTTPトランザクションオブジェクトに格納します。

  • "Host": ホスト情報を格納し、CGI用に使用します。
  • "Content-Length": コンテンツの長さを格納します。
  • "Transfer-Encoding" と値が "chunked" の場合: チャンクエンコーディングが使用されていることを示すフラグを設定します。
  • "Content-Type" と値が "multipart/form-data" を含む場合: マルチパートフォームデータが使用されていることを示すフラグを設定します。

std::string & key,
std::string & value,

HTTPヘッダーラインを解析し、キーと値を抽出します。

この関数はHTTPヘッダーラインを解析し、コロン（':'）を使用してキーと値に分割します。 キーはコロンの前の文字列、値はコロンの後の文字列です。値の前後の空白はトリムされます。 特に、"Content-Type" ヘッダーの場合は、追加でセミコロン（';'）による解析を行い、 マルチパートフォームデータのバウンダリを抽出します。

Parameters
line解析するヘッダーライン。
key抽出されたキーを格納する文字列への参照。
value抽出された値を格納する文字列への参照。
boundary抽出されたバウンダリ文字列を格納する文字列への参照（"Content-Type" ヘッダーの場合）。
Note
  • 例: "Host: example.com" は key="Host", value="example.com" として解析されます。
