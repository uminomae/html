void ContentExtractionState::handle ( )
virtual

HTTPリクエストに基づいてコンテンツ抽出処理を実行します。

有効なファイルパスを探索し、ファイルの内容をHTTPレスポンスボディに読み込みます。 ファイルが見つからない場合は、404エラーの処理を行います。
