void DeleteMethodState::handle ( )
virtual

Delete method の場合を処理します。

Note
  • 動作は実行済みで、更なる情報は給されない場合 ：204 (No Content)
  • 方針: なるべくならbodyは却下する方針で実装する
    DELETE 要請メッセージのペイロードには、意味論は定義されない — 既存の実装には、ペイロード本体を伴って送信されてきた DELETE 要請を却下するものもある。
