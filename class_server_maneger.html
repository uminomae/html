◆ runServerLoop()

void ServerManeger::runServerLoop ( )

サーバーのメインループを実行します。

例外が発生した場合はサーバーのループを再起動します。
HOT_RELOAD_CONFIGが定義されている場合、設定のホットリロードの初期化を行います。

Note
一度のpollで二つのフラグを立てることはできないので、一度のループでPOLLINとPOLLOUTをそれぞれ見ます。

◆ setupServers()

void ServerManeger::setupServers ( )

設定ファイルを解析(parse)し、サーバーごとのポートでlistenを行い、pollfdを準備します。

設定ファイル(.conf, mime.types)をパースして内部設定に反映させた後、サーバー指定ディレクティブごとにソケットを設定し、リスニングを開始します。 最後に、サーバーのソケットをノンブロッキングモードに切り替えます。

Note
DEBUG_THROW_SETUPが定義されている場合、セットアップ時に意図的に例外を投げてプログラムを終了します（デバッグ目的）。
ドキドキするとコードがプリティになります。
