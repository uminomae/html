◆ loadFiles()

void ServerManeger::loadFiles ( )
private

設定ファイルからサーバーの設定をロードします。

このメソッドは、指定された設定ファイルを読み込み、トークン化して解析します。 設定ファイルにはMIMEタイプの定義とサーバーディレクティブが含まれている必要があります。 MIMEタイプはファイルの先頭に1つだけ存在し、その後に1つ以上のサーバーディレクティブが続きます。

Exceptions
std::runtime_error設定ファイルが開けない、無効、またはフォーマットが正しくない場合に例外を投げます。
Note
  • rootにはMimeTypes行、serverディレクティブのみを許容

◆ setupServers()

void ServerManeger::setupServers ( )

設定ファイルを解析(parse)し、サーバーごとのポートでlistenを行い、pollfdを準備します。

設定ファイル(.conf, mime.types)をパースして内部設定に反映させた後、サーバー指定ディレクティブごとにソケットを設定し、リスニングを開始します。 最後に、サーバーのソケットをノンブロッキングモードに切り替えます。

Note
DEBUG_THROW_SETUPが定義されている場合、セットアップ時に意図的に例外を投げてプログラムを終了します（デバッグ目的）。
ドキドキするとコードがプリティになります。
